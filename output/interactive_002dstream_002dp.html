Function

interactive-stream-p

interactive-stream-p stream ⇒ generalized-boolean

Arguments and Values

stream—a stream.

generalized-boolean—a generalized boolean.

Description

Returns true if stream is an interactive stream; otherwise, returns false.

Examples
